# regaloPerfetto
App regalo perfetto che ti guida nel trovare l'idea giusta per il tuo regalo

## Avvio locale

Questo progetto e' una web app React + Vite. Non va aperto con Expo Go.

1. Installa le dipendenze con `npm install`
2. Avvia il server con `npm run dev`
3. Apri l'app nel browser all'indirizzo mostrato in terminale, oppure dal telefono usando l'IP locale del computer
